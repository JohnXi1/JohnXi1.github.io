Kobe Bryant’s Shot Selection Final Report

Ⅰ.Introduction

The main topic of the research is to predict and evaluate the field goal percentage of Kobe Bryant, a former NBA all-star shooting guard. Since there are only two results of a shot——making or missing, and each shot attempt is relatively independent, this problem naturally falls into the two- class problems of Machine learning. In order to provide enough data for us to analyze, my data set includes every shot Kobe had attempted throughout his entire career. The data set is retrieved from NBA stats. My intended objectives of this research are to find out the major factors that lead to high shooting percentage and find new ways of analyzing basketball stats. They are of significance since they can not only improve my basketball skills but also give me information about machine learning and modeling.

From my point of view, the factors that influence Kobe’s shooting percentage most are the distance between Kobe and the hoop and the shooting type. Normally, the closer you get to the basket, the higher field goal percentage you will get. Also, in many cases, the lay-ups are more accurate than mid-range jumpers and three-pointers. To test my hypothesis, I need to utilize some models to solve classification problems,among which the decision tree is the most famous and plain. Also, Confidence Weighted Linear Classification is also a good model that is widely utilized by mathematicians in solving classification problems.

III.Methods

The main method we are using in this topic is feature engineering. The major goal of taking this method is that we need to crab the most crucial and relative data from the entire dataset and leave out one that has limited eﬀect to our study.

We would go through three steps as follows——

  • Grouping numerical data to categorical data.

  • Encoding categorical attributes.

In this step, we use feature matrix to make it more eﬃcient.

  • Feature selection.

With analysis and coding, we found out that the best number of features we can get is 3. Too many or less features would cause the result to be misleading, as figure 5 shows.

Figure5

Also, for this step, we incorporate some basic algorithms. They can shows the importance the diﬀerent categories of data can contribute to the final estimation model. The first one is Recursive Feature Elimination, or RFE for short. Random Forest, Extra Trees and Gradient Boosting classifier were used for the RFE, and there are total 21 features selected in the feature engineering. Also, we use the Information Entropy, which can help us calculate the numerical result about whether the feature consists of enough information.

Then, we can select the features that we consider beneficial to the results and use them to train our model. Finally, we can presume the Kobe’s final shooting percentage.

What’s more, within the code, there can be two diﬀerent models that can help us get the parameters as accurate as possible. The first one is the decision tree. It is an algorithm which utilizes incremental structure and tons of classifiers to divide the data into smaller groups with similar traits. In order to increase its accuracy, mathematicians have developed its upgraded version——random forest. Instead of just using one decision tree, random forest contains many trees with diﬀerent criteria. And after all trees come out the answers, random forrest deduces the final outcome by majority-voting. The second one is called boosting. Boosting is used to advance the weaker learners into stronger ones. It is fundamental strategy is to divide the elements for a few times until there exists one pattern that can divide data with similar traits into certain groups.

IV.Results

After feature selection, training the model, and optimizing the parameters, the model can give out the solution that the Kobe’s average shooting percentage should be around 45.2 percent after all. And the code can export a csv file which includes the estimated shooting percentage of Kobe for the shots that we left out for testing. And

comparing to the real data, our shot percentage correctness reaches 60.148 percent, which is a pretty outstanding number.

V.Conclusion

In this project, I used random forest and boosting to predict Kobe’s shooting percentage. I find the feature of the model by Recursive Feature Elimination. I test several machine learning models by radiant boosting and cross validation. Finally I used Random Forest, Gradient Boosting, Extra Trees, Extreme Gradient, and Boosting to make the prediction. Looking into the future, for better results, more complex models and algorithms can be used to further improve the results. Also, increasing the size of the dataset is another way to improve the results.
